print output:

        for (var i = 0; i < 3; i++) {
            setTimeout(function log() {
              console.log(i); // What is logged?
            }, 1000);
          }

output:3 3 3

Explanation:

As above code runs as asynchronous i.e, the loop and setTimeout func, for loop doesn't wait for setTimeout() to complete and continous its loop - there i value will be 0,1,2,3.but the func(setTimeout) has been called for 3 times in the execution-context.thus the i present updated value will be printed(3 times as per loop condition).
